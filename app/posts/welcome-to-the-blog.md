# Welcome to the Blog!

This is your first blog post. Congratulations on getting your blog up and running!

## What You Can Do Now

Your blog is fully functional with:

- **Static Site Generation** - Lightning-fast page loads
- **Dynamic Comments** - Real-time comment system with Firebase
- **Newsletter Subscription** - Subscribe readers to your mailing list
- **Admin Dashboard** - Full content management system
- **GitHub Integration** - Write posts directly in GitHub and auto-deploy

## Getting Started

1. Visit the admin dashboard at `/admin/login`
2. Enter your admin password
3. Create new posts using the dashboard or push markdown files to GitHub
4. Your posts will be live within minutes!

## Features

### For Readers
- Fast, responsive blog interface
- Comment on posts
- Subscribe to newsletter
- Dark mode support

### For Admins
- Create, edit, delete posts
- Moderate comments
- Manage subscribers
- View dashboard statistics
